Capabilities
What makes this portfolio-worthy

Skills that enterprise AI teams actually hire for.

Multi-agent orchestration
LangGraph StateGraph with 6 nodes sharing typed state. Each agent is independently testable and swappable.
Retrieval-augmented generation
Category-filtered vector retrieval over a local knowledge base. Hit-rate improved from 78% to 92% with filtering.
Hallucination detection
LLM-as-judge evaluates every draft against retrieved context before it reaches a human. Catches ~10% of cases.
Human-in-the-loop
No response auto-sends. Every draft goes through approve / edit / reject. Production-safe by design.
Smart escalation logic
Rule-based escalation for angry sentiment, critical urgency, or zero KB matches — no LLM tokens wasted.
Full observability
Agent trace visible on every ticket. LangSmith integration for production debugging and latency analysis.
